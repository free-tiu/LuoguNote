## 题目描述
输入一个小写字母，输出其对应大写字母。例如输入q[回车]时，会输出Q

### 提交代码
#### C
```
#include <stdio.h>

int main() {
    int a;
    scanf("%c",&a);
    if('a'<=a&&a<='z')a-='a'-'A';
    printf("%c", a);
    return 0;
}
```

-----

#### Python
   _在看了题解之后，才发现用python来解决可能更快。_ 
   _在python中，`upper()`是专门将字符串中小写字母转换成大写字母；`lower()`则是将大写转换成小写_
```
s = input()
print(s.upper())
```
